# [p0kks.me](https://p0kks.me)

Minimal, fast personal portal with a brutalist style. Built for speed and simple content delivery.

## 🚀 Features

- **Design**: High-contrast, grayscale, typography-focused.
- **Content**: Loads portfolio and notes from GitHub Issues (REST API).
- **Performance**:
  - No frameworks. No build step.
  - Session caching for API calls.
  - Animations pause when not visible.
- **UI**: Glass-style elements using `backdrop-filter`.
- **Responsive**: Works on mobile and desktop.
- **Background**: Animated Nordic rune matrix (respects reduced motion).
- **Accessibility**: Semantic HTML, ARIA, SEO + social metadata.

## 🛠 Tech

- **Frontend**: HTML, CSS, JavaScript (ES6+)
- **Libraries**:
  - Marked.js (Markdown)
  - DOMPurify (sanitization)
  - Highlight.js (code highlighting)
- **CMS**: GitHub Issues
